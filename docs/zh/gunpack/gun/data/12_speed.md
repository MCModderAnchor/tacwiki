# 移动速度

本页介绍枪械重量和移速修正的配置。

## 重量 `weight`

**可选**，数值类型。枪械重量（kg），影响持有者的移动速度。默认每千克减少 1.5% 移动速度，该比例可在 ServerConfig 中调整。

```json
"weight": 3.5
```

::: tip
`weight` 作为 `MULTIPLY_BASE` 类型的属性修饰器应用于实体，实际对移速的影响较为复杂，并非简单的百分比扣除。
:::

## 移速修正 `movement_speed`

**可选**，对象类型。在不同状态下对移动速度做额外修正。多种情况同时满足时，仅计算最后一个匹配的修正。

```json
"movement_speed": {
  "base": 0.0,
  "aim": -0.2,
  "reload": -0.15
}
```
此例来自默认枪包 `m4a1_data.json`。

| 键        | 说明      |
|----------|---------|
| `base`   | 默认移速修正  |
| `aim`    | 瞄准时移速修正 |
| `reload` | 换弹时移速修正 |

::: tip
`movement_speed` 作为 `MULTIPLY_TOTAL` 修饰器应用，枪械和各配件的修正值会先求和后再应用。
:::
