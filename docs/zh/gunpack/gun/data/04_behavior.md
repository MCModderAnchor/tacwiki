# 枪械行为

本页介绍枪栓类型、拉栓上膛时长以及趴下/斜握行为的配置。

## 枪栓类型 `bolt`

**必填**，枚举类型。决定枪械的枪栓工作方式，会影响枪械模型中特定分组的显示控制。

| 值               | 说明                                               |
|-----------------|--------------------------------------------------|
| `closed_bolt`   | 闭膛待击。例如开火时会隐藏模型中 `bullet_in_barrel` 分组，枪机复位后恢复显示 |
| `open_bolt`     | 开膛待机。例如模型中 `bullet_in_barrel` 分组始终隐藏             |
| `manual_action` | 手动上膛，每次射击后需要手动拉栓才能继续开火                           |

```json
"bolt": "closed_bolt"
```

## 拉栓上膛时长 `bolt_action_time`

**可选**，数值类型。仅当 `bolt` 为 `manual_action` 时有效，单位为秒。指定每次射击后拉栓动作的时长。

```json
"bolt_action_time": 0.9
```

## 趴下 `can_crawl`

**可选**，布尔值。是否允许持枪时趴下，默认 `true`。

```json
"can_crawl": true
```

## 斜握 `can_slide`

**可选**，布尔值。是否允许斜握（仅视觉效果），默认 `true`。

```json
"can_slide": true
```

::: tip
斜握在玩家潜行（蹲下）时触发，仅改变持枪姿态的视觉表现，不影响任何数值。
:::
