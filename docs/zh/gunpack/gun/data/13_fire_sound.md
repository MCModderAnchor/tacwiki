# 音效

本页介绍射击声音传播距离的配置。

## 射击声音 `fire_sound`

**可选**，对象类型。控制射击声音的传播距离。两个乘数均基于全局设置中的默认距离计算。

```json
"fire_sound": {
  "fire_multiplier": 1.0,
  "silence_multiplier": 1.0
}
```

| 键                    | 说明                           |
|----------------------|------------------------------|
| `fire_multiplier`    | 无消音器时射击声音传播距离的乘数，值越小枪声传播范围越小 |
| `silence_multiplier` | 安装消音器后射击声音传播距离的乘数            |

此例为默认值，大多数枪械不单独配置此项。

### 消音器与生物仇恨

游戏内枪声会吸引特定范围内的敌对生物（即使未发现玩家）和中立生物（如末影人、僵尸猪灵）。安装消音器后：

- 枪声传播距离受 `silence_multiplier` 缩放，影响玩家听到的声音范围。
- 携带消音器开火时，枪声不再吸引敌对生物和中立生物，但生物原本的仇恨机制（发现玩家后产生仇恨）依然生效。
- 以上两部分是独立的机制：声音传播距离是给玩家听的，仇恨吸引是给生物听的。两者均可被全局设置调整。
