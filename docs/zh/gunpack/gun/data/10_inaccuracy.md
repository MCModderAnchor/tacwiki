# 精度与晃动

本页介绍各种姿态下的精准度和受伤晃动倍率的配置。

## 精准度 `inaccuracy`

**可选**，对象类型。定义不同姿态下的子弹散布（数值越小越精准）。

```json
"inaccuracy": {
  "stand": 4.5,
  "move": 5,
  "sneak": 2.5,
  "lie": 1.5,
  "aim": 0.15
}
```  
此例来自默认枪包 `ak47_data.json`。  


| 键       | 说明   |
|---------|------|
| `stand` | 站立散布 |
| `move`  | 移动散布 |
| `sneak` | 潜行散布 |
| `lie`   | 趴下散布 |
| `aim`   | 瞄准散布 |

### 精度状态优先级

五种精度状态**互斥**，每次射击仅取**一个**状态的值。优先级从高到低为：

| 优先级 | 状态 | 触发条件 |
|-------|------|---------|
| 1 | `aim` | 完全瞄准（瞄准进度 = 100%） |
| 2 | `lie` | 趴下 |
| 3 | `sneak` | 潜行（蹲下） |
| 4 | `move` | 移动中 |
| 5 | `stand` | 站立（无以上任何状态） |

只有瞄准进度达到 **100%** 时才会切换到 `aim` 散布。在瞄准过渡期间（进度 < 100%），仍使用当前姿态（站立/潜行/趴下）的散布值。

::: tip
`aim` 散布不是叠加在其他状态之上，而是在完全瞄准时**替换**当前状态的散布。例如同时趴下和瞄准时，射击取 `aim` 的 0.15，而非 `lie` 的 1.5。
:::

## 受伤晃动 `hurt_bob_tweak_multiplier`

**可选**，数值类型。受伤时镜头晃动的倍率。1 为原版晃动程度，0 为完全不晃动，默认 0.05。

```json
"hurt_bob_tweak_multiplier": 0.05
```

默认值 0.05 意味着受伤时晃动大幅减弱。
