# 精度与晃动

本页介绍各种姿态下的精准度和受伤晃动倍率的配置。

## 精准度 `inaccuracy`

**可选**，对象类型。定义不同姿态下的子弹散布（数值越小越精准）。

```json
"inaccuracy": {
  "stand": 4.5,
  "move": 5,
  "sneak": 2.5,
  "lie": 1.5,
  "aim": 0.15
}
```  
此例来自默认枪包 `ak47_data.json`。  


| 键       | 说明   |
|---------|------|
| `stand` | 站立散布 |
| `move`  | 移动散布 |
| `sneak` | 潜行散布 |
| `lie`   | 趴下散布 |
| `aim`   | 瞄准散布 |

::: tip
散布值越大，子弹落点越分散。不同姿态的散布独立计算，`aim` 瞄准散布是在瞄准状态下额外应用的基础散布。
:::

## 受伤晃动 `hurt_bob_tweak_multiplier`

**可选**，数值类型。受伤时镜头晃动的倍率。1 为原版晃动程度，0 为完全不晃动，默认 0.05。

```json
"hurt_bob_tweak_multiplier": 0.05
```

默认值 0.05 意味着受伤时晃动大幅减弱，这是 TACZ 枪械的默认行为。
