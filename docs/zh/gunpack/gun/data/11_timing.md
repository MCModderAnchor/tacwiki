# 动作时长

本页介绍抬枪、收枪、瞄准和跑射延迟的时长配置。

## 抬枪时长 `draw_time`

**可选**，数值类型。枪械抬起的动作时长（秒）。抬起动作完成后，枪械才能进行开火、检视等行为。

```json
"draw_time": 0.35
```

## 收枪时长 `put_away_time`

**可选**，数值类型。枪械收起的动作时长（秒）。

```json
"put_away_time": 0.57
```

## 瞄准时长 `aim_time`

**可选**，数值类型。从腰射切换到瞄准状态的过渡时长（秒）。

```json
"aim_time": 0.2
```

## 跑射延迟 `sprint_time`

**可选**，数值类型。从冲刺状态切换到可以开火的延迟时长（秒）。

```json
"sprint_time": 0.2
```

::: tip
抬枪和收枪的时长需要与动画文件中的动作时长匹配。抬枪时长可以短于动画，枪械在时长结束后即可操作，动画继续播放；收枪时长短于动画则可能导致动画表现异常。
:::
