# 弹药

本页介绍枪械的弹药类型、弹匣容量和扩容弹匣的配置方式。

## 弹药类型 `ammo`

**必填**，字符串类型。指定枪械使用的弹药，格式为 `命名空间:弹药ID`。

```json
"ammo": "tacz:762x39"
```

弹药 ID 对应枪包 `data/{命名空间}/index/ammo/` 目录下的弹药注册文件。默认枪包提供的弹药包括：

| 弹药ID              |  名称              |
|-------------------|------------------|
| `tacz:9mm`        | 9mm 手枪弹          |
| `tacz:45acp`      | .45 ACP 手枪弹      |
| `tacz:50ae`       | .50 AE 手枪弹       |
| `tacz:357mag`     | .357 马格南 手枪弹     |
| `tacz:22wmr`      | .22 温彻斯特 马格南 手枪弹 |
| `tacz:500mag`     | .500 马格南 手枪弹     |
| `tacz:762x25`     | 7.62x25mm 手枪弹    |
| `tacz:57x28`      | 5.7x28mm 穿甲弹     |
| `tacz:46x30`      | 4.6x30mm 穿甲弹     |
| `tacz:556x45`     | 5.56x45mm 步枪弹    |
| `tacz:545x39`     | 5.45x39mm 步枪弹    |
| `tacz:762x39`     | 7.62x39mm 步枪弹    |
| `tacz:762x54`     | 7.62x54mm 步枪弹    |
| `tacz:58x42`      | 5.8x42mm 步枪弹     |
| `tacz:68x51fury`  | 6.8x51mm 步枪弹     |
| `tacz:308`        | .308 温彻斯特 步枪弹    |
| `tacz:300blk`     | .300BLK 步枪弹      |
| `tacz:6x43`       | 6x43mm 步枪弹       |
| `tacz:792x57`     | 8mm 毛瑟 步枪弹       |
| `tacz:30_06`      | .30-06 春田 步枪弹    |
| `tacz:45_70`      | .45-70"政府"步枪弹    |
| `tacz:338`        | .338 拉普马格南 狙击弹   |
| `tacz:50bmg`      | .50 BMG 狙击弹      |
| `tacz:12g`        | 12 号口径霰弹         |
| `tacz:40mm`       | 40mm 榴弹          |
| `tacz:rpg_rocket` | RPG-7 火箭弹        |

自定义弹药需要在 `data/{命名空间}/index/ammo/` 下创建注册文件，并在 `assets/{命名空间}/display/ammo/` 下提供显示文件（包括模型、贴图、抛壳、曳光弹颜色等）。

::: tip
弹药本身不决定伤害——伤害、速度、穿透等属性由枪械 data 中的 `bullet` 字段控制。同一种弹药用在不同枪械上可以有不同的弹道表现。
:::

## 弹匣容量 `ammo_amount`

**必填**，数值类型。枪械标准弹匣的装弹数。

```json
"ammo_amount": 30
```

::: warning
此值适用于除背包直读（`reload.type` 为 `inventory`）以外的所有供弹模式。背包直读模式的弹药上限由背包内存放的弹药数量决定，详见[换弹](02_reload)。
:::

## 扩容弹匣 `extended_mag_ammo_amount`

**可选**，数组类型。扩容弹匣的弹药容量，按等级从低到高排列，最多包含 3 个元素。

```json
"extended_mag_ammo_amount": [34, 37, 40]
```

- 数组索引与扩容等级一一对应：`[0]` 对应一级扩容，`[1]` 对应二级，`[2]` 对应三级。只填写你的扩容弹匣配件实际用到的等级对应的位置即可。
- 扩容弹匣配件通过 `allow_attachment_types` 中的 `extended_mag` 槽位启用，详见[配件槽](15_attachment)。
