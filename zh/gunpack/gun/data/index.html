枪械数据进阶

枪械数据配置（Data）

本页列出枪械 xxx_data.json 的所有可用字段，详情见各小节。

字段总览

弹药

字段类型必填说明详情
ammo字符串是指定弹药类型，如 tacz:762x39弹药
ammo_amount数值是弹匣容量弹药
extended_mag_ammo_amount数组否扩容弹匣容量，按等级从低到高弹药
reload.type枚举是供弹方式，可选 magazine / manual / fuel / inventory供弹
reload.infinite布尔值否无限备弹，与 inventory 模式不兼容供弹
reload.feed.empty数值否空仓换弹子弹补充时间点（秒）供弹
reload.feed.tactical数值否战术换弹子弹补充时间点（秒）供弹
reload.cooldown.empty数值否空仓换弹总耗时（秒）供弹
reload.cooldown.tactical数值否战术换弹总耗时（秒）供弹

逻辑脚本

字段类型必填说明详情
script字符串否指定枪械逻辑脚本，位于 data/{命名空间}/scripts逻辑脚本
script_param对象否传给脚本的参数，内容取决于脚本需要逻辑脚本

枪械行为与射速

字段类型必填说明详情
bolt枚举是枪栓类型，可选 open_bolt / closed_bolt / manual_action枪械行为
bolt_action_time数值否拉栓上膛时长（秒），仅 manual_action 有效枪械行为
can_crawl布尔值否是否允许趴下，默认 true枪械行为
can_slide布尔值否是否允许斜握，默认 true枪械行为
rpm数值是每分钟射速，最大 1200开火机制
fire_mode数组是开火模式列表，可选 semi / auto / burst开火机制
fire_mode_adjust对象否不同开火模式下的数值修正（伤害、射速、精度等）开火机制
burst_data对象否连发模式数据（仅在 fire_mode 含 burst 时调用）开火机制
charging对象否蓄力/延迟扳机配置开火机制

子弹实体属性

字段类型必填说明详情
bullet.life数值是子弹存活时间（秒）子弹实体属性
bullet.damage数值是单发基础伤害子弹实体属性
bullet.bullet_amount数值否霰弹弹丸数，默认 1子弹实体属性
bullet.speed数值是子弹初速度（m/s），游戏内默认全局弹速乘2子弹实体属性
bullet.gravity数值否重力系数子弹实体属性
bullet.friction数值否空气阻力子弹实体属性
bullet.knockback数值否击退效果子弹实体属性
bullet.pierce数值否穿透数子弹实体属性
bullet.ignite布尔值/对象否是否点燃实体或方块子弹实体属性
bullet.ignite_entity_time数值否点燃实体时间（秒），默认 2子弹实体属性
bullet.explosion对象否爆炸配置（半径、伤害、是否破坏方块等）子弹实体属性
bullet.tracer_count_interval数值否曳光弹间隔，0 为每发都是曳光弹，无此字段不发射子弹特效

额外伤害

字段类型必填说明详情
extra_damage.armor_ignore数值否护甲穿透率，默认 0额外伤害
extra_damage.head_shot_multiplier数值否爆头伤害倍率额外伤害
extra_damage.damage_adjust数组否距离衰减表，超过最远距离伤害为 0额外伤害

后坐力

字段类型必填说明详情
recoil.pitch数组否纵向后坐力曲线（time/value 关键帧）后坐力
recoil.yaw数组否横向后坐力曲线（time/value 关键帧）后坐力
crawl_recoil_multiplier数值否趴下后坐力倍率后坐力

精度与晃动

字段类型必填说明详情
inaccuracy.stand数值否站立散布精度与晃动
inaccuracy.move数值否移动散布精度与晃动
inaccuracy.sneak数值否潜行散布精度与晃动
inaccuracy.lie数值否趴下散布精度与晃动
inaccuracy.aim数值否瞄准散布精度与晃动
hurt_bob_tweak_multiplier数值否受伤摇晃倍率，默认 0.05精度与晃动

动作时长

字段类型必填说明详情
draw_time数值否抬枪动作时长（秒）动作时长
put_away_time数值否收枪动作时长（秒）动作时长
aim_time数值否瞄准时长（秒）动作时长
sprint_time数值否跑射延迟（秒）动作时长

移动速度

字段类型必填说明详情
weight数值否枪械重量（kg），影响移速移动速度
movement_speed.base数值否默认移速修正（MULTIPLY_TOTAL 修饰器）移动速度
movement_speed.aim数值否瞄准时移速修正移动速度
movement_speed.reload数值否换弹时移速修正移动速度

音效传播

字段类型必填说明详情
fire_sound.fire_multiplier数值否射击声音传播距离乘数音效
fire_sound.silence_multiplier数值否消音射击声音传播距离乘数音效

枪械近战

字段类型必填说明详情
melee.distance数值否近战攻击范围，与配件加和枪械近战
melee.cooldown数值否近战冷却时间（秒），与配件加和枪械近战
melee.default.animation_type枚举否动画类型，可选 melee_push / melee_stock枪械近战
melee.default.damage数值否近战伤害枪械近战
melee.default.knockback数值否近战击退枪械近战
melee.default.range_angle数值否近战攻击范围角度枪械近战
melee.default.prep数值否近战前摇时长（秒）枪械近战

配件槽

字段类型必填说明详情
allow_attachment_types枚举数组否开放的配件槽，可选 scope / stock / muzzle / grip / laser / extended_mag配件槽
builtin_attachments对象否原厂配件候选列表及默认预装件配件槽
exclusive_attachments对象否为特定配件指派独立属性（⚠ 功能待修复）配件槽

过热

字段类型必填说明详情
heat.max数值否过热上限过热
heat.per_shot数值否每发子弹过热值过热
heat.cooling_multiplier数值否冷却倍率过热
heat.cooling_delay数值否冷却延迟（ms）过热
heat.over_heat_time数值否过热后延迟（ms）过热
heat.min_inaccuracy数值否过热对精度的最小影响倍率过热
heat.max_inaccuracy数值否过热对精度的最大影响倍率过热
heat.min_rpm_mod数值否过热对射速的最小影响倍率过热
heat.max_rpm_mod数值否过热对射速的最大影响倍率过热
